셰프랑반찬 메뉴 사진 넣는 곳
================================

이 폴더(assets/img/)에 아래 파일명으로 사진을 저장하면
해당 메뉴 사진이 자동으로 교체됩니다.
(파일이 없으면 기존 키워드 사진 → 이모지 순으로 자동 대체되어 화면은 깨지지 않습니다.)

- 권장 형식: JPG (png도 가능하나 .jpg 확장자로 저장 권장)
- 권장 크기: 가로 600px 이상, 정사각형(1:1) 또는 4:3
- 파일명은 아래 영문 그대로 (대소문자 구분)

사용법
------
1) 위 파일명으로 사진을 이 폴더에 저장
2) 저장 후 알려주시면 라이브 사이트(GitHub Pages)에 반영(배포)합니다.

* 파일명을 다르게 쓰셨다면, "파일명 = 메뉴명"만 알려주셔도 제가 맞춰서 연결합니다.
* 다른 메뉴(달력 전체) 사진도 같은 방식으로 추가 가능합니다. 말씀해 주세요.
